# GenAI Engineering Portfolio

A 30-day structured sprint building toward a **Code Review Agent** —
a tool that accepts a Python file, reviews it for bugs and anti-patterns
with grounded reasoning, and exposes a FastAPI endpoint.

**Weeks 1:** LLM fundamentals  
**Week 2:** RAG layer (knowledge base for code best practices)  
**Week 3:** Agent layer (multi-step reasoning)  
**Week 4:** Evaluation + deployment
