# GenAI Engineering Portfolio

A 30-day structured sprint building toward DevContext AI - a developer assistant that ingests a codebase, company docs, and DB schemas into a unified knowledge base. 
It answers onboarding questions ("how does auth work?"), and assesses PR impact ("what breaks if I change this?"). 
Exposes a FastAPI endpoint. Built with RAG, agentic tool use, and incremental re-ingestion via GitHub webhooks.


**Weeks 1:** LLM fundamentals  
**Week 2:** RAG layer (knowledge base for code best practices)  
**Week 3:** Agent layer (multi-step reasoning)  
**Week 4:** Evaluation + deployment
